# Shape Area Calculations 📐

This repository contains Python programs for calculating the areas of various 2D and 3D shapes. Each shape with the necessary code to calculate its area.

## Shapes Included
- Circle
- Cone
- Cube
- Cuboid
- Cylinder
- Octahedron
- Rectangle
- Sphere
- Square
- Tetrahedron
- Trapezoid
- Triangle
